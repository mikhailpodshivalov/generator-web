Fast loop pipeline

2. Randomize Instrument Sound

FAQ

How long is the demo?

Demo period is 7 days, starting after first successful activation.

How many devices per license?

Current target policy is up to 3 devices per key.

What if payment is done but key is delayed?

Keep the same email and retry status check. The order flow is designed for webhook delay retries.

Where should I start learning?

Open First 5 minutes, then continue with Random Workflow.

Is MG subscription-based?

No subscription. MG is licensed directly, and random generation is not limited.

Is MG lightweight to run?

MG is a native desktop app written in Rust and optimized for low CPU and memory usage.

Who is behind MG?

Masterpiece Generator is built and maintained as an independent solo project.
