This is not a DAW — it's an idea generator.

Randomize, tweak by hand, then use your ideas to create full masterpieces.

  • 1. Hit any button — get instant music (Scratch mode)
  • Chaos — randomness freedom, best above 50% for more variety
  • Euclid — spreads notes evenly across a row, keep under 25% for musicality
  • Random whole pattern (all tracks at once):
  • Super rand — everything + BPM, root note & scale
  • Pattern rand — notes & steps only
  • Instr rand — instruments & synth settings
  • FX rand — delay, reverb, filter, drive
  • P-lock rand — per-step parameter locks
  • Single track row: pattern / sound / fx / p-lock for one row
  • Or tweak by hand: place notes, ⚙ for synth, right-click step for slide / p-lock
  • Tip: slots 7–8 can be any instrument — swap hats for leads and get wild combos
  • 2. Save what you like (Bank mode)
  • Save to Bank — keep pattern in a bank slot
  • 99 bank slots, each holds a pattern and a song
  • Yellow dot = has pattern
  • Green dot = has song
  • Save the whole bank to a file, export pattern as MP3/WAV via Render
  • 3. Generate a full song (Song mode)
  • Auto-generates a full song from your saved bank pattern
  • Build Song — auto-arranges intro → buildup → drop → outro
  • Edit any part, re-randomize individual patterns freely
  • Export full song as MP3, WAV, or MIDI for your DAW
  • 4. For pros: MIDI out
  • Route generated patterns to hardware or software synths via MIDI
  • Assign a MIDI channel per instrument + up to 4 CC locks per step
